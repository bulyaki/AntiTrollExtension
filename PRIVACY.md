# Privacy Policy — TrollMarker

**Last updated:** March 16, 2026

## Summary

TrollMarker does **not** collect, store, or transmit any personal data.

## Data Storage

All user data — including your watch lists, subscription URLs, and color preferences — is stored **locally on your device** using the browser's built-in storage (`browser.storage.local`). This data never leaves your device.

## Network Requests

The extension may fetch publicly accessible text files from URLs you configure (e.g., GitHub Gists). These requests are made directly from your browser to the specified servers. No data from your browser or Facebook account is sent during these requests.

## Third-Party Services

This extension does **not** use any analytics, tracking, advertising, or third-party data collection services.

## Facebook Data

The extension reads the contents of Facebook pages you visit **only** to identify and highlight matching profile links. It does not store, log, or transmit any Facebook content.

## Open Source

This extension is open source and its full source code is publicly available for review.

## Contact

If you have questions about this privacy policy, please open an issue on the project's GitHub repository.
